# Methodology

1. **Ingest**: Load EFA approvals, HSSD budgets, deseg transfers
2. **Link**: Match students to transfers and EFA usage
3. **Drain**: Calculate per-pupil loss vs. state average
4. **Compare**: Hot Spring vs. Saline (control)
5. **Output**: Visuals, CSVs, briefing PNGs

> Raw data in `raw/` (never edited)  
> Scripts in `src/`  
> Outputs in `out/`
